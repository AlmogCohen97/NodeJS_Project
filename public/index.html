Furniture Api documentation

Hey, I`m Almog Cohen and this is my Node-JS Project of furniture api that I've written. 2022

Furnitures Routes

Get information about item via a full API

Method URL Description
Get /furnitures/all?page=number Get a list of all the furnitures , present 10 results by page number
Get /furnitures/all?perPage=number Get X items in each page
Get /furnitures/all?reverse=sort-key Reverse by sort key: yes/no
Get /furnitures/all?sort=key-word From which value to sort by
Get /furnitures/prices?min=number&max=number Price range
Get /furnitures/category/:category-name Get all furnitures from the same category
Get /furnitures/search?s=keyword Search products by name ,color or info
Get /furnitures/myshop/:{userId} Get all furnitures of your User only. Must to be connected!
